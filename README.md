## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/krishh21/AI-image-Genrator.git

# Install dependencies
npm install

# Add your STABILITY_API_KEY to a .env file
STABILITY_API_KEY=your_api_key_here

# Run locally
npm run start
